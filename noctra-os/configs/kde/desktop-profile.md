# KDE Desktop Profile

Target: KDE Plasma with a macOS Ventura-inspired layout.

## Baseline

- top panel with application menu, active-window title, system tray, clock
- bottom centered dock-style panel with pinned applications
- window buttons on the left
- Breeze base theme with Noctra branding until a custom theme is packaged
- Papirus or custom Noctra icon set
- SDDM Noctra login theme in a later milestone

## Pinned Applications

- Firefox or LibreWolf
- Dolphin
- Konsole
- System Settings
- KeePassXC
- Wireshark

## Packaging Note

Ventura-like third-party themes should be reviewed for license and maintenance risk before shipping in a commercial webshop product. Prefer a Noctra-owned theme derived from permissive components.
